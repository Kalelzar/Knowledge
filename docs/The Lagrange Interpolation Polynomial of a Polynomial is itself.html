The Lagrange Interpolation Polynomial of a Polynomial is itself

Interpolation Lagrange Numeric Methods Theorem

If \(f \in \pi_{n}\) then \(L_{n}(f;x) = f(x)\)

1 The Lagrange Interpolation Polynomial of a Polynomial is itself

If \(f \in \pi_{n}\) then \(L_{n}(f;x) = f(x)\)

On both sides there are polynomials from \(\pi_{n}\) which interpolate the same table.

x0 ⋯ xn
f(x0) ⋯ f(xn)

From the uniqueness of the Lagrange interpolation problem \(\Rightarrow L_{n}(f;x) = f(x)\)
